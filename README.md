# Objectifs

Le but de ce projet est de réaliser un RPG en ligne de commande en utilisant des language orienté objet.
* Dresser un diagramme de classes
* Mettre en place une application en ligne de commande
* Développement avec TypeScript
* Utilisation des concepts de classe et d'instance
* Maîtrise du concept d'héritage et de polymorphisme
* Utilisation des concepts OO intermédiaires (écrasement de méthodes, interfaces, classes abstraites...)

# Debut du projet

* Reflexion sur le theme et les classe du jeu.
* -> Le joueur entre dans un donjon Mythologique et devra afronter des personalitées des Mythologies, Nordique, Grec et Egyptienes.
